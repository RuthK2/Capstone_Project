EXPENSE TRACKER API - SIMPLE PROJECT PLAN
==========================================

PROJECT OVERVIEW
-----------------
A simple expense tracking API where users can register, login, and manage their personal expenses.

MAIN FEATURES
-------------
1. User Authentication
   - User registration
   - User login/logout
   - Secure authentication with JWT tokens

2. Expense Management
   - Add new expenses
   - View all expenses
   - Update existing expenses
   - Delete expenses

3. Categories
   - Predefined categories: Groceries, Electricity, Utilities, Miscellaneous, Electronics, Clothing
   - Organize expenses by category

4. Basic Filtering
   - Filter expenses by time periods
   - Filter expenses by categories
   - Simple expense summaries

API ENDPOINTS
--------------
Authentication:
- POST /api/auth/register
- POST /api/auth/login
- POST /api/auth/logout

Expenses:
- GET /api/expenses
- POST /api/expenses/create
- PUT /api/expenses/:id/update
- DELETE /api/expenses/:id/delete
- GET /api/expenses/summary

Categories:
- GET /api/categories
